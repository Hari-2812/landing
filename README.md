# AI/Data Analytics Landing Page

## Tech
- React (functional components)
- Tailwind CSS
- Framer Motion

## Folder Structure

```text
src/
  components/
    layout/Container.jsx
    ui/Icon.jsx
    ui/SectionHeader.jsx
  data/content.js
  sections/
    HeroSection.jsx
    OverviewSection.jsx
    ExpertsSection.jsx
    RolesSection.jsx
    RoadmapSection.jsx
    CurriculumSection.jsx
    ProjectsSection.jsx
    BenefitsSection.jsx
    EnrollmentSection.jsx
    TestimonialsSection.jsx
    AchievementsSection.jsx
  App.jsx
  index.css
  main.jsx
```

## Integrate your images
Place your assets in:

```text
public/media/
  experts/expert-1.jpg ...
  projects/project-1.jpg ...
  testimonials/user-1.jpg ...
  achievements/achievement-1.jpg ...
```

Update `src/data/content.js` paths/names if your filenames differ.

## Run
```bash
npm install
npm run dev
```
